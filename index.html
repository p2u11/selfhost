The victim

Our tortured victim is an old plastic MacBook that I've got for ~$30. (dirt cheap AND useless!)
Previous victim, a MacBook Pro Mid-2012, was unfortunately sold because I need money. It was the most powerful laptop I could use for a project like this, unfortunately the sale led to a huge downgrade in terms of power. I'm sorry.

Specs:

  • CPU: Intel Core 2 Duo T8300 @ 2.40GHz
  • RAM: 3 GB
  • Storage: 120 GB HDD for the OS, 1 TB SSD for data
  • OS: Debian 13.4.0 (Trixie)
  • Other specs aren't important cuz its a server now

As for the current condition, it has a cracked case from all sides, the battery is completely dead and it can only run on AC power, the cooling system is clogged with dust and the CPU thermal paste is probably dried out.
Due to some modern SATA bullshit, my 60 GB SSD doesn't work here, so I'm using a 120 GB HDD for the system and a 1 TB SSD for data storage, which is pretty sad but it is what it is.

I actually used it as a daily driver for a few weeks back in Jan 2025 on Windows Vista, and it was surprisingly usable for basic tasks like web browsing and media consumption, though I didn't even try anything more demanding than that. It ran Novetus (a launcher for old Roblox clients) pretty well. Mind you, my Surface Pro 3 runs Austiblox worse than that, so it was pretty impressive for a 15 year old laptop.
So what can I say, the laptop's still alive and we better use it at least as a server until it kicks the bucket.
